外协管理工具 - Win7 离线使用说明

1. 将整个“外协系统”文件夹复制到 Win7 电脑本地磁盘，建议放在 D:\外协系统 或 C:\外协系统。
2. 双击 start-server.bat 启动程序。
3. 在浏览器中打开 http://127.0.0.1:8765 使用。
4. 如果需要局域网其他电脑访问，使用启动窗口显示的局域网地址。
5. 业务数据保存在 data\outsourcing-data.json，迁移或备份时请保留 data 文件夹。

说明：
- 本文件夹已内置 Node.js 12.22.12，无需在 Win7 上单独安装 Node.js。
- 不建议直接在 \\服务器\共享目录 这类 UNC 路径中运行。可以复制到本机磁盘，或先把共享目录映射成盘符后再运行。
- 建议 Win7 使用 Chrome 109 或 Firefox ESR 等较新的离线浏览器。
- 如果无法访问，请检查 Windows 防火墙是否允许 8765 端口或 node.exe。
